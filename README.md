# LinuxStudy
这是学习Unix系统编程时做的一些小练习。

Unix下众多API要想全部记住对我来说不太现实，所以挑选了其中一些比较重要的函数进行练习。
为了使用方便，所有练习都使用了C++11进行包装，但是底层还是使用的Unix系统Api，
项目管理使用的是 `cmake` ，所以建议使用 CLion 进行查看和编译。
